📥 Import Soal Massal

Unggah file Excel untuk memindahkan ratusan soal ke dalam database dengan cepat.

💡 Info Mesin Import V2:

  1. Mendukung otomatis 4 Tipe Soal: PG, PG Kompleks, Matriks (B-S), & Esai.
  2. Bisa menyisipkan kode Rumus (LaTeX) dan memanggil Link Gambar otomatis.
  3. Mendukung Mode Hybrid: Kosongkan sel di Excel jika Anda ingin copas soal bergambar rumit (Snipping Tool) dari Word ke Web Editor nanti.
  4. Sangat disarankan membaca petunjuk di Sheet 2 pada file template.
